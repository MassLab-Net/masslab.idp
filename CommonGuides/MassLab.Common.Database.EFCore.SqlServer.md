# MassLab.Common.Database.EFCore.SqlServer

Use this for EF Core repositories backed by SQL Server.

NuGet:

```bash
dotnet add package MassLab.Common.Database.EFCore.SqlServer
```

Typical config section: `Database` with `WriteConnectionString`, optional `ReadConnectionString`, and `UseSeparateReadDb`.

Use this package when the service uses SQL Server with EF Core.

Read deeper only when needed: `MassLab.Common.Database.EFCore.SqlServer/README.md`.
