# MassLab.Common.Database.EFCore

Use this for EF Core repository and unit-of-work implementations.

NuGet:

```bash
dotnet add package MassLab.Common.Database.EFCore
```

Use provider-specific packages for application registration:

- `MassLab.Common.Database.EFCore.PostgreSQL`
- `MassLab.Common.Database.EFCore.SqlServer`
- `MassLab.Common.Database.EFCore.MySQL`

Provides EF Core repositories, query extensions, specification evaluator, and save-change interceptors.

Read deeper only when needed: `MassLab.Common.Database.EFCore/README.md`.
