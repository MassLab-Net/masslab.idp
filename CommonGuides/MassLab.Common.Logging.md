# MassLab.Common.Logging

Use this for common logging abstractions and options.

NuGet:

```bash
dotnet add package MassLab.Common.Logging
```

Main types:

```csharp
ILoggerAdapter<T>
LoggingOptions
```

Registration API in README may use legacy naming. Prefer Serilog package for actual structured logging setup when the app needs production logging.

Read deeper only when needed: `MassLab.Common.Logging/README.md`.
